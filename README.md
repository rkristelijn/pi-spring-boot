# pi-spring-boot
Tutorial on setting up a simple Spring Boot (Java Web) project on a Rapsberry Pi

# Raspbian
I have downloaded the Raspberry Pi Desktop [2018-06-27-rpd-x86-stretch.iso](http://downloads.raspberrypi.org/rpd_x86/images/) and installed it using VirtualBox on my laptop. Then I used the out of the box cd mount for the guest additions to fully utalize the screen resources.
